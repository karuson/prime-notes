Terminal Shortcuts

cd > change directory
cd .. > directory above current
cd / > root level directory
cd ~ > home folder (~ = tilde)
cd ~/directory > specify directory to go to

tab > finishes what I'm typing
pwd > print working directory (where am I?)
pwd|pbcopy > copy working directory to clipboard

ls > list folders/files in current directory
mkdir > new folder
touch > new file [ touch new-file.file-type ]
mv > move/rename
rm > delete file
rm -R > delete directory
code . > opens project in code editor
open [ file ] > opens file
*** up arrow key can be used to go to last command run in the terminal ***

Git & GitHub

Git is a way to track changes that you make to code
GitHub is library of our saved code, portfolio

Configuration

git --version //check git version
git config --global user.name "" //set user name
git config --global user.email "" //set email with Git signup email address
git config --global init.defaultBranch "main" //naming branch main

Process

Fork //make copy of the repo (assignment) on GitHub > on Prime GitHub page select fork to your account from the menu bar, make sure it shows your username afterwards
Clone //bring copy to local computer > open the forked project, select code SSH (make sure you are at your forked version), copy URL
Change //write and edit code
Stage //select files to save
Commit //finalize save changes to local computer

git clone [ copied SSH URL ] //clone forked project to local computer
git status //shows changes
git add . //stages files to be ready for commit(save)
git commmit -m "" //finalizes changes with message
git push origin main //sends updated commit to GitHub

JavaScript

value = assignment

string variables // text or string of characters
number variables // constant numbers
boolean variables // true or false

const = constant declaration > doesn't change
